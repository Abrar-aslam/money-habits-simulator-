## ğŸ§  Analytics & â€œAI-styleâ€ Logic

The app uses straightforward logic to mimic AI-like behavior while staying deterministic and explainable:

1. **Summaries**
   - Aggregates all transactions to compute:
     - Total income.
     - Total expenses.
     - Net balance.
     - Savings rate (net / income).

2. **Trend-based Prediction**
   - Groups expenses by month.
   - Uses a simple **linear trend** over the last few months:
     - Calculates slope and intercept.
     - Predicts next-month expense.
   - Clamps negative predictions to zero and uses the predicted value to:
     - Generate â€œtrend up/downâ€ commentary.
     - Suggest creating buffers or keeping habits.

3. **AI-style Text Insights**
   - Maps numeric metrics to human-friendly sentences, for example:
     - Savings rate > 30% â†’ â€œexcellentâ€.
     - Savings rate 10â€“30% â†’ â€œgood but improvableâ€.
     - Savings rate < 10% â†’ â€œneeds attentionâ€.
   - Identifies top spending category and suggests trimming it.
   - Shows â€œsince last visitâ€ changes based on stored previous summary.

4. **Habits**
   - For each habit:
     - Checks last completion date.
     - If completed today: keep streak.
     - If completed yesterday: streak++.
     - Otherwise: streak resets to 1 on new completion.
   - Reflects current streaks in the UI with simple icons and colors.

These approaches are inspired by how AI-augmented finance tools and habit trackers provide insights and gamification. [web:77][web:89][web:91]
